# RP-checker

Статическая страница для проверки RP-ников в формате `Имя_Фамилия`.

## Что внутри

- Проверка формата ника
- Поиск запрещенных слов
- Подсказки при ошибках
- Генератор случайного ника
- История последних проверок
- Генератор краткого личного дела персонажа
- Готовность к публикации на GitHub Pages

## Как запустить локально

Открой `index.html` в браузере или подними простой статический сервер.

Пример с Python:

```bash
python -m http.server 8000
```

После этого открой:

```text
http://localhost:8000
```

## Как залить на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи туда файлы `index.html`, `styles.css`, `script.js`.
3. Открой `Settings` -> `Pages`.
4. В `Source` выбери ветку `main` и корень `/root`.
5. Сохрани настройки и дождись публикации.

Ссылка будет выглядеть примерно так:

```text
https://yourname.github.io/rp-checker/
```

## Как встроить на форум

Если форум разрешает `iframe`, можно вставить так:

```html
<iframe
  src="https://yourname.github.io/rp-checker/"
  width="100%"
  height="900"
  style="border:0;border-radius:16px;"
  loading="lazy">
</iframe>
```

Если `iframe` запрещен, просто вставляй прямую ссылку на страницу.
